# Sanjana Projects Portfolio

A professional portfolio website showcasing my work as a "Vibe Coder". Built with HTML, CSS, and vanilla JavaScript, served via a simple Node.js Express server.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop and mobile.
- **Projects Page**: A dedicated "Hold On !" animated page for projects.
- **Contact Integration**: 
  - Email popup modal with copy-to-clipboard functionality.
  - Direct LinkedIn and GitHub icon links.
- **Interactive UI**:
  - Smooth scrolling navigation.
  - Animated hero section text.
  - Custom cursor and hover effects.

## Setup & Running

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start the Server**:
   ```bash
   npm start
   ```

3. **View the Site**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `public/index.html`: Main landing page.
- `public/projects.html`: Dedicated projects page.
- `public/style.css`: Main stylesheet.
- `public/modal.css`: Styles for the contact popup.
- `public/script.js`: Interactive logic (navigation, modal, etc).
- `server.js`: Simple Express server configuration.

## Customization

- **Images**: Located in `public/images/` and the root directory.
- **Content**: Update HTML files in `public/`.
